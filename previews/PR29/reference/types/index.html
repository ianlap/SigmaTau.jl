Shared types

Core data types used across the package and re-exported from SigmaTau.

AbstractTimingData

Supertype of timing data records (PhaseData, FrequencyData).

SigmaTau.PhaseData — Type
PhaseData{T<:AbstractFloat}

Phase residuals x(t) sampled at uniform interval tau0.

  • x::Vector{T} where T<:AbstractFloat: Phase samples in seconds.

  • tau0::Float64: Base sample interval τ₀ in seconds.

source
SigmaTau.FrequencyData — Type
FrequencyData{T<:AbstractFloat}

Fractional-frequency samples y(t) at uniform interval tau0.

  • y::Vector{T} where T<:AbstractFloat: Fractional-frequency samples (dimensionless).

  • tau0::Float64: Base sample interval τ₀ in seconds.

source
SigmaTau.StabilityResult — Type
StabilityResult

Unified return type for every stability calculation.

The noise_type, ci_lower, ci_upper, and edf vectors are empty when the calculation was invoked with calc_ci=false.

  • deviation_type::Symbol: Which deviation produced this result (e.g. :adev, :mdev).

  • tau::Vector{Float64}: Analysis intervals τ in seconds.

  • dev::Vector{Float64}: Stability deviation σ_y(τ) per interval.

  • noise_type::Vector{Symbol}: Noise-type symbol identified at each τ (empty unless calc_ci=true).

  • ci_lower::Vector{Float64}: Lower CI bound (empty unless calc_ci=true).

  • ci_upper::Vector{Float64}: Upper CI bound (empty unless calc_ci=true).

  • edf::Vector{Float64}: Equivalent degrees of freedom (empty unless calc_ci=true).
