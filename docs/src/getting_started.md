# Getting Started

## Installation

SigmaTau.jl is a workspace monorepo with three subpackages re-exported
through an umbrella `SigmaTau`. From the Pkg REPL:

```julia-repl
pkg> add https://github.com/ianlap/SigmaTau.jl
```
