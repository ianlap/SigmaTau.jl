# Publications

A curated reading list, organized by the role each work plays in
SigmaTau's foundation. For the alphabetical reference index, see the
[Bibliography](../bibliography.md).

## Foundational

- **[Riley & Howe 2008](@cite riley-2008-sp1065)** — *Handbook of
  Frequency Stability Analysis*, NIST Special Publication 1065. The
  canonical reference for the Allan / Modified-Allan / Total / Hadamard
  family definitions, bias factors, and EDF approximations that
  SigmaTau's Stab submodule reproduces.
- **[IEEE 1139-2022](@cite ieee1139-2022-definitions)** — *Standard
  Definitions of Physical Quantities for Fundamental Frequency and Time
  Metrology*. The current normative source for stability-statistic
  notation, units, and reporting conventions.
