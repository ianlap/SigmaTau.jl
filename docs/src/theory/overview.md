# Theory: Overview

This chapter covers the power-law noise model used throughout SigmaTau.jl,
the relationship between phase and fractional-frequency data, and the
defining time-domain stability variances.

Content fills in across follow-up PRs from `legdocs/handbook/` and
`legdocs/equations/`.

## Outline

- Phase residuals `x(t)` vs fractional frequency `y(t)`
- The five power-law noise types (WPM, FPM, WFM, FFM, RWFM) and slope table
- Sample interval τ₀ and analysis interval τ = mτ₀
- σ²_y(τ) family overview
