# Theory: Allan Family

Allan / Modified Allan / Hadamard / Time / Lapinski deviations.

Content fills in across follow-up PRs.

## Outline

- ADEV: σ²_y(τ) = ½⟨(ȳ_{i+1} - ȳ_i)²⟩
- MDEV: phase-averaged second difference
- HDEV: third-difference variant, robust to linear drift
- TDEV: τ · MDEV / √3
- LDEV: third-difference modified-style estimator (IEEE 1139-2022 Annex C)
